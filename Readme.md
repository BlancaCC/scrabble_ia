# "Inteligencia artificial" para jugar a formar palabras a partir de una serie de letras  

Autora: Blanca Cano Camarero 
Fecha: Enero 2019  

## documentación 
En el directoro doc, encontrará un fichero documentacion.md con las ideas esenciales para 
entender la construcción del programa. 

Además todo ha sido comentado con doxygen lo cual puede ser bastante esclaracedor. 

## make 
`make ` para generar todos los programillas
`make juego_longitud` o `make juego_puntos`
`make doxy` Genera documentación doxygen y abre firefox con esta
`make clear ` Borra módulos y documentación

## make test 
Todos los módulos disponenen de un test de prueba puede invocarlo con:
- make testdiccionario  
- make testletras  
- make testcombinaciones  
- make testbiblioteca  
- make testia  

Todos los módulos contienes más funcionalidades de las utilizada en el juego, para poder verlas en todo su explendero y comprenderlas mejor ejecutar los test.
